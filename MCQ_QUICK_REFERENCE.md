# MCQ Generation Quick Reference

## Format Template

```
[DEFINITION WITHOUT ANSWER TERM] is called ________.

A. [Correct Answer]
B. [Distractor 1]
C. [Distractor 2]
D. [Distractor 3]
```

---

## Transformation Formula

```
INPUT:
Topic Title = "Photosynthesis"
Definition = "Process by which plants convert light energy into chemical energy"

STEP 1: Remove answer term from definition
"Process by which plants convert light energy into chemical energy"

STEP 2: Add "is called ________." ending
"The process by which plants convert light energy into chemical energy is called ________."

STEP 3: Set correct answer
Correct Answer: "Photosynthesis"

STEP 4: Add distractors
Distractors: ["Cellular Respiration", "Transpiration", "Osmosis"]
```

## Quality Checks

✅ Answer term NOT in question  
✅ Ends with "is called ________."  
✅ Definition is complete  
✅ Grammatically correct  
✅ Distractors are plausible  

---

## Common Mistakes to Avoid

❌ "The carbon cycle is important." → Answer leaked  
❌ "A process in ecosystems." → No "is called" format  
❌ "This thing does X." → Too vague  

✅ "A process where carbon circulates between organisms and environment is called ________."
